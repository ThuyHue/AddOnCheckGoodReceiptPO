1. Nội dung: Kiểm tra số lượng cho phép nhập vào kho không được nhỏ hơn hoặc vượt 
quá số lượng order +(-) số lượng chỉ định max(min) khi thiết lập Enterprise
2. Điều kiện tiên quyết 
Truy cập theo đường dẫn để tạo UDF:  Administration > Application Setup > User Defined Fields. 
- Object Type: chọn	Enterprise
- Field Name: Đặt U_PerentOfQuantity
- Check vào: Display in Results(Cho phép trường tạo hiển thị trong Enterprise)
- Mở Enterprise nhập giá trị giới hạn dùng để tính toán
